# Aide Skills LeRobot

Liste les skills disponibles et explique comment les utiliser.

## Skills disponibles

- `/lerobot-status` : Diagnostic systeme complet (ports serie, manette, cameras, moteurs, espace disque)
- `/lerobot-xbox` : Lance le controle du robot via la manette Xbox
- `/lerobot-teleop` : Lance la teleoperation (leader/follower)
- `/lerobot-record` : Lance l'enregistrement de donnees (dataset)
- `/lerobot-calibrate` : Lance la calibration des moteurs
- `/lerobot-cameras` : Verifie les cameras connectees
- `/lerobot-help-skills` : Cette aide

Affiche cette liste a l'utilisateur de maniere claire et concise.
